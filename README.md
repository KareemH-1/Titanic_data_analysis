# Titanic Data Analysis

This project analyzes the Titanic dataset using Python. It includes data processing, visualization, and an interactive dashboard.

## Libraries Used

The following Python libraries are used in `titanic.py`:

- numpy
- pandas
- matplotlib
- seaborn
- dash
- plotly

## Files

- `Titanic.csv`: The dataset file.
- `titanic.py`: The main analysis and dashboard script.

## How to Run

1. Install the required libraries:
   ```bash
   pip install numpy pandas matplotlib seaborn dash plotly
   ```
2. Run the script:
   ```bash
   python titanic.py
   ```

## Description

- Data is loaded and analyzed using pandas and numpy.
- Visualizations are created with matplotlib, seaborn, and plotly.
- An interactive dashboard is built using Dash.
